# 店舗サイトテンプレート運用マニュアル

このプロジェクトは、飲食店・美容室・整体院・小規模店舗向けに使い回せる営業用ホームページテンプレートです。

## 1. 店舗ごとに変更する場所

### index.htmlで変更する項目

| 項目 | 変更内容 |
|---|---|
| title | Google検索やブラウザタブに表示されるタイトル |
| meta description | Google検索に表示される説明文 |
| og:title | SNS共有時のタイトル |
| og:description | SNS共有時の説明文 |
| og:image | SNS共有時の画像 |
| 店名 | ヘッダー・フッター・店舗情報 |
| キャッチコピー | Heroセクションの大きな見出し |
| サブコピー | Heroセクションの説明文 |
| 電話番号 | tel: から始まるリンクをすべて変更 |
| 住所 | Accessセクション |
| 営業時間 | Accessセクション |
| Google Map | iframeのsrcを変更 |
| メニュー名 | Menuセクション |
| メニュー説明 | Menuセクション |
| 口コミ文 | Reviewsセクション |

## 2. 画像の差し替えルール

画像は `images` フォルダに入れます。

| ファイル名 | 用途 |
|---|---|
| hero.webp | トップの大きな画像 |
| concept.webp | 店舗・雰囲気紹介画像 |
| menu01.webp | メニュー1 |
| menu02.webp | メニュー2 |
| menu03.webp | メニュー3 |
| gallery01.webp | ギャラリー1 |
| gallery02.webp | ギャラリー2 |
| gallery03.webp | ギャラリー3 |
| gallery04.webp | ギャラリー4 |
| gallery05.webp | ギャラリー5 |
| gallery06.webp | ギャラリー6 |

## 3. 他店舗へ流用するときの手順

1. プロジェクトフォルダをコピーする
2. フォルダ名を店舗名に変更する
3. images内の画像を差し替える
4. index.htmlの店名・電話番号・住所・営業時間を変更する
5. Google Mapを変更する
6. ChromeでPC表示とスマホ表示を確認する
7. GitHubへpushする
8. GitHub Pagesで公開する

## 4. 業種別の変更ポイント

### 飲食店
- Hero画像は料理写真
- CTAは「電話予約」
- Menuセクションを重視

### 美容室
- Hero画像は店内・施術写真
- CTAは「予約する」
- Featureは「技術・空間・カウンセリング」

### 整体院
- Hero画像は院内・施術風景
- CTAは「相談する」
- Featureは「痛み改善・姿勢・安心感」

### 工務店・会社サイト
- Hero画像は施工事例・外観
- CTAは「無料相談」
- Menuは「サービス内容」に変更

## 5. 営業で見せるときの説明例

「こちらは飲食店向けに制作したデモサイトです。スマホ対応・Google Map・電話予約導線・写真ギャラリーまで実装しています。御社用に写真・文章・色味を差し替えて、店舗の雰囲気に合わせたホームページとして制作できます。」

## 6. 注意点

実店舗の写真・ロゴ・口コミを使う場合は、公開前に必ず店舗側の許可を取ってください。
営業用デモとして見せるだけの場合でも、無断で公式サイトのように公開しないようにしてください。
