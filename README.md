# 草鍋 喜多呂 デモサイト

大阪・九条の飲食店「草鍋 喜多呂」を想定した営業用デモサイトです。

## ファイル構成

```text
kitaro-restaurant-demo/
├── index.html
├── style.css
├── script.js
├── images/
├── icons/
├── README.md
├── robots.txt
└── sitemap.xml
```

## 画像の置き場所

`images` フォルダに以下の名前で画像を入れてください。

```text
hero.webp        トップ画像
concept.webp     店内写真
menu01.webp      草鍋
menu02.webp      エビマヨ
menu03.webp      小籠包
gallery01.webp   外観
gallery02.webp   草鍋アップ
gallery03.webp   店内
gallery04.webp   料理集合
gallery05.webp   エビマヨ
gallery06.webp   小籠包
```

## 店舗情報の変更場所

`index.html` の以下を編集してください。

- 店名
- キャッチコピー
- 住所
- 営業時間
- 電話番号
- Google Map
- メニュー内容

## 公開方法

GitHub Pagesを使うと無料で公開できます。

1. GitHubにpush
2. リポジトリの Settings を開く
3. Pages を開く
4. Branch を `main` にする
5. Save
6. 表示されたURLを営業先に共有

## 注意

このサイトは営業用デモです。
実店舗の公式サイトとして公開する場合は、店舗様の許可を取ってください。
